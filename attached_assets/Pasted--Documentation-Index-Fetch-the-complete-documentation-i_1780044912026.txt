# Hyper MCP

> Bring Hyper's integrations, built-in tools, skills, and platform into Claude, ChatGPT, OpenCode, and any AI app that supports MCP.

<iframe width="100%" height="410" src="https://www.youtube.com/embed/NZLAdOL9fP8" title="Hyper MCP" frameBorder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowFullScreen />

The Hyper MCP server brings everything you can do on Hyper — your app integrations, built-in tools, skills, and native platform features — into any AI app that supports the [Model Context Protocol](https://modelcontextprotocol.io). One endpoint, one auth, hundreds of tools.

That includes:

* **Your app integrations** — Gmail, Shopify, HubSpot, Slack, Notion, Google Drive, and the rest of the [56+ integrations](/integrations/overview) you've connected to Hyper
* **Ad platforms** — Meta, Google Ads, TikTok, LinkedIn, Pinterest, and others
* **Hyper's built-in scrapers and generators** — Meta Ads Library, Reddit, Twitter, Instagram, TikTok scrapers; image generation; video generation
* **Hyper's native platform** — create and manage agents, tasks, skills, files, knowledge bases, and tables from anywhere

Instead of stitching together dozens of custom MCP servers and managing 40+ API keys, point your AI app at one endpoint and you've got the whole Hyper toolbox.

## What you can do

Two things, end to end:

**1. Use Hyper's tools inside another AI app.** Run scrapers, query your integrations, generate images, browse the web — without leaving Claude, ChatGPT, or wherever you work.

**2. Drive the Hyper platform from another AI app.** Create agents, assign tasks, attach skills, build tables, and manage files in your Hyper workspace by chatting with Claude or ChatGPT. The agents and tasks you build show up in Hyper, ready to run on schedule or in response to webhooks.
